## üíª Local Development Setup

### Prerequisites

- Python 3.9 or higher
- pip (Python package installer)

### Installation Steps

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd gsrec
   ```

2. **Create and activate a virtual environment:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run Database Migrations:**
   ```bash
   python3 manage.py migrate
   ```

5. **Run the Development Server:**
   ```bash
   python3 manage.py runserver
   ```
   Visit `http://127.0.0.1:8000/` in your browser.

## üåê Deployment (cPanel / Production)

This project is configured to work with WhiteNoise for serving static files in production environments like cPanel (using Passenger or Python App Setup).

### Deployment Steps:

1. **Collect Static Files:**
   ```bash
   python3 manage.py collectstatic --noinput
   ```

2. **Environment Variables:**
   Ensure you set `DEBUG=False` and a secure `SECRET_KEY` in your production environment settings.

3. **Dependencies:**
   Make sure `whitenoise` is installed in your production environment.

## üìÅ Project Structure

- `core/`: Main app containing templates, views, models, and static files.
- `gsrec_project/`: Project configuration, settings, and URLs.
- `static/`: Root directory for shared static assets.
- `media/`: User-uploaded files (logos, country banners, test prep images).
- `staticfiles/`: Compiled static files for production (generated by `collectstatic`).
